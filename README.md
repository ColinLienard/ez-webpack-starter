Basic Wepack starter using Sass, Typescript and more.

## Installation

Prefer Yarn to NPM.

```bash
# Yarn
yarn

# NPM
npm i
```

## Use

### Development

Create a development server with hot realoding.

```bash
# Yarn
yarn dev

# NPM
npm dev
```

### Production

Create a `build` folder with minified and optimized files.

```bash
# Yarn
yarn build

# NPM
npm build
```
